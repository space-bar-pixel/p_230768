สามารถดาวน์โหลดได้จาก Python.org

บันทึกโปรแกรม:

เปิด Text Editor (เช่น VS Code หรือ Notepad++)

คัดลอกโค้ดไปวางในไฟล์ใหม่

บันทึกไฟล์เป็น temperature_converter.py

รันโปรแกรม:

เปิด Terminal/Command Prompt และไปที่โฟลเดอร์ที่บันทึกไฟล์ไว้

รันโปรแกรมด้วยคำสั่ง:

bash
คัดลอก
แก้ไข
python temperature_converter.py
