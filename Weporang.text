โปรแกรมที่คุณเขียนขึ้นคือ เกมทอยลูกเต๋า ที่ให้ผู้เล่น 2 คนแข่งกันทอยลูกเต๋าคนละ 3 ครั้ง แล้วดูคะแนนรวม ใครได้คะแนนเยอะที่สุดจะชนะ

การใช้งาน
เริ่มเกม: เมื่อโปรแกรมเริ่มทำงาน, ผู้เล่นทั้งสองจะถูกขอให้ทอยลูกเต๋าคนละ 3 ครั้ง

กด Enter เพื่อให้ผู้เล่น 1 ทอยลูกเต๋า

กด Enter อีกครั้งเพื่อให้ผู้เล่น 2 ทอยลูกเต๋า

คะแนนสะสม: คะแนนจะถูกเก็บสะสมจากการทอยแต่ละครั้ง

แสดงผล: เมื่อผู้เล่นทอยครบ 3 ครั้งแล้ว โปรแกรมจะแสดงคะแนนรวมของทั้งสองผู้เล่น

ผู้ชนะ: ระบบจะประกาศผลว่าใครได้คะแนนมากที่สุด หรือถ้าเสมอกัน

วิธีเล่น
เปิดโปรแกรมใน Python environment หรือ IDE เช่น VS Code หรือ PyCharm

เมื่อโปรแกรมเริ่มทำงาน มันจะถามให้กด Enter เพื่อให้แต่ละผู้เล่นทอยลูกเต๋า

ทอยครบ 3 ครั้งแล้ว จะประกาศผลการแข่งขัน
